# Smart City Traffic Navigation System

## Overview
Smart City Traffic Navigation System is a C++ application that helps users find the most efficient route between locations in a city. The project uses Dijkstra's Algorithm to calculate the shortest path and provides a graphical interface using Raylib.

## Features
- Shortest path calculation using Dijkstra's Algorithm
- Interactive city map visualization
- Route selection between different locations
- Distance and path display
- User-friendly graphical interface
- Efficient graph-based navigation system

## Technologies Used
- C++
- STL (Standard Template Library)
- Raylib
- Visual Studio Code

## Algorithm
The project implements Dijkstra's Algorithm to find the shortest path between nodes in a weighted graph representing city roads and intersections.

## Project Structure
- Graph Representation
- Dijkstra Algorithm Implementation
- User Interface using Raylib
- Route Visualization Module

## Requirements
- C++ Compiler (GCC/MinGW)
- Raylib Library
- Visual Studio Code

## How to Run
1. Install Raylib.
2. Clone or download the project.
3. Open the project in Visual Studio Code.
4. Compile the source files.
5. Run the executable file.
